Sahar 0.1.19 release bundle

Changes in this build:
- improved Linux distribution detection in install.sh, install_master.sh, and install_agent.sh
- switched installer detection logic to use ID and ID_LIKE from /etc/os-release instead of brittle PRETTY_NAME matching
- added clearer detected OS and init-system output during installation
- kept supported installer families explicit: Alpine, Debian, and Ubuntu-compatible systems
